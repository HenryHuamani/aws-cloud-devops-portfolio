# Laboratorio 01 - AWS IAM

## Objetivo

Aprender el funcionamiento de AWS Identity and Access Management (IAM).

## Servicios utilizados

- IAM

## Actividades realizadas

- Creación de usuario IAM
- Creación de grupo Administrators
- Asignación de permisos AdministratorAccess
- Activación de MFA
- Inicio de sesión con usuario IAM

## Evidencias

Las capturas del laboratorio se encuentran en la carpeta **evidencias**.

## Lecciones aprendidas

- No utilizar la cuenta Root para tareas diarias.
- Administrar permisos mediante grupos y políticas.
- Habilitar MFA para mejorar la seguridad.
- Aplicar el principio de mínimo privilegio.
