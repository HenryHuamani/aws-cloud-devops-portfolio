
# AWS IAM Architecture

This document describes the architecture implemented during **Lab 01 – Identity & Access Management (IAM)**.

## Architecture Overview

The diagram represents the logical relationship between the main IAM components used during the laboratory.

The implementation includes:

- IAM Service
- AdministratorAccess managed policy
- Administrators group
- IAM user (henry.admin)
- Virtual MFA device
- AWS Management Console authentication
