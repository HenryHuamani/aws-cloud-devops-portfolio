# Commands – Amazon EC2

## Connect using SSH

```bash
ssh -i henry-key.pem ubuntu@PUBLIC_IP
```

---

## Show current user

```bash
whoami
```

---

## Display hostname

```bash
hostname
```

---

## Current directory

```bash
pwd
```

---

## List files

```bash
ls
```

---

## Update Ubuntu

```bash
sudo apt update

sudo apt upgrade -y
```

---

## Check Linux version

```bash
cat /etc/os-release
```

---

## Show CPU information

```bash
lscpu
```

---

## Memory information

```bash
free -h
```

---

## Disk usage

```bash
df -h
```

---

## Running services

```bash
systemctl
```
